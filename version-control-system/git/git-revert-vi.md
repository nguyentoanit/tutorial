# Git Manual Rollback
## Tổng quan
Tài liệu này mô tả hướng dẫn rollback source code trên git bằng command line. Việc rollback không xóa commit cũ.

## Yêu cầu
Tải và cài đặt Git ([download](https://git-scm.com/downloads)) trên máy.
